# チェンジログ

このプロジェクトのすべての注目すべき変更は、このファイルに記録されています。

## [1.1.0] - 2026-07-05

### 追加

#### HOME / END キーの機能実装

- エクスプローラー部ではサムネセレクターを先頭・最後尾に移動させる
- ビューアー部ではアーカイブファイル内の現在地を先頭・最後尾に移動させる

#### ビューアー部で右クリックメニューから現在の見開き設定を保存できる

- シングルページモードは保存できないが、上書き保存でシングルページモードもメニュー選択は出来るようにはなっている。セーブなしのデフォルト値に戻すだけのアクション
- 綴じ方向、オフセット値も保存される
- ページ位置は保存せずファイル先頭で状態復元される。オフセット値は破壊しないような復元をするので位置補正用の仮想ページが現れることがあるがバグではない
- データベースファイルは実行ファイル横に配置される。初期化したいときは消してください

#### サムネイルバーにページ番号の追加

#### アーカイブ形式の追加対応

- TAR / CBT（無圧縮・gzip圧縮）に対応。`.tar` / `.cbt` / `.tgz` / `.tar.gz`
- TAR + zstd圧縮（`.tar.zst` / `.tzst`）に対応。純Rust実装でC依存の追加なし

#### 画像形式の追加対応

- TIFF / TIF に対応（純Rust実装）。マルチページTIFFは1ページ目のみ表示、CCITT G3/G4やJPEG-in-TIFFなど一部の圧縮方式は非対応

### 修正

- 拡張子のファイルタイプ判定からファイルの冒頭のバイナリ列をみてデコードや展開を行うようにした。これによりデコードできないファイルに遭遇したときのフォールバック不可バグの修正もあわせてできている
- サムネイルバーの表示が重かったのを修正

### 検討中

- RAR形式アーカイブ対応
- お気に入りフォルダ対応：お気に入りファイルを仮想お気に入りフォルダに格納していく仕様
- 拡大縮小機能

## [0.5.0] - 2026-06-26

### 追加
- ディレクトリナビゲーションがシングルクリック化されました（[直]ボタンは削除されました）
- 無効なZIPマーカーキャッシュ: 画像を含まないZIPは`.neko/invalid/`の下にマークされます。ZIPが更新されると自動的にマーカーは削除されます
- 無効なZIPはグリッド内に赤いXで表示され、ダブルクリック時にトースト通知（3秒後に自動消滅）が表示されます
- ビューアで前後にナビゲートする際、無効なZIPは黙ってスキップされます
- サムネイル保存がJPGの個別ファイルから単一のデータベースファイルにリファクタリングされました

### 変更
- ディレクトリツリーの拡張は▶をクリックすることでのみ実行されるようになりました（アイテムをシングルクリックするとディレクトリ内に移動します）

### 修正
- ビューアは画像エントリを含まないZIPに対して開かなくなりました。無効としてキャッシュされると、エクスプローラーから非表示になります

### 進行中
- UI多言語対応（i18n）のサポート
