# チェンジログ

このプロジェクトのすべての注目すべき変更は、このファイルに記録されています。

## [0.7.0] - 2026-06-30

### 追加
- 多言語対応 : 切り替えメニューをサムネイル表示部のメニューバー右側に配置した(ja/en/cn)
- (主にwindows)F5 から F8キーのウィンドウサイズ / ポジション 記憶復帰機能にデフォルト番号をconfにて割り振れるようにした

### 変更
- キャッシュの最適化

### 変更 (開発メモ)
- 基本構造をMVCモデル化
- マルチウィンドウの更新不具合でeframeの除外

### 既知の問題
- キャッシュメモリあふれ時の表示不具合、特に巨大なアニメーションファイルの対応。もしくは表示しきれない場合のフォールバック
- ローディング表示の明確化
- 設定ファイルの多言語化

### 作業中
- 検索機能
- アーカイブ内移動中のサムネイルと移動マーカー

## [0.6.1] - 2026-06-27

### 追加
- `←` / `→` キーでのファイルジャンプ: 現在のページ位置に関係なく、前後のファイルの1ページ目に直接ジャンプします

### 変更
- RAMキャッシュは、ファイルごとに縦スクロールと横スクロールの領域に分けて管理するようになりました。ファイル間のブラウジング時のキャッシュ効率が向上します

## [0.5.0] - 2026-06-26

### 追加
- ディレクトリナビゲーションがシングルクリック化されました（[直]ボタンは削除されました）
- 無効なZIPマーカーキャッシュ: 画像を含まないZIPは`.neko/invalid/`の下にマークされます。ZIPが更新されると自動的にマーカーは削除されます
- 無効なZIPはグリッド内に赤いXで表示され、ダブルクリック時にトースト通知（3秒後に自動消滅）が表示されます
- ビューアで前後にナビゲートする際、無効なZIPは黙ってスキップされます
- サムネイル保存がJPGの個別ファイルから単一のデータベースファイルにリファクタリングされました

### 変更
- ディレクトリツリーの拡張は▶をクリックすることでのみ実行されるようになりました（アイテムをシングルクリックするとディレクトリ内に移動します）

### 修正
- ビューアは画像エントリを含まないZIPに対して開かなくなりました。無効としてキャッシュされると、エクスプローラーから非表示になります

### 進行中
- UI多言語対応（i18n）のサポート

## [0.4.0] - 2026-06-26

### 追加
- Shift+スクロール / Shift+↑↓でのファイルナビゲーション: 最初/最後のページにいるときに前後のファイルに移動します
- Linux ローカルビルドサポート: `setup.sh`と`Makefile`を追加し、ガイド付きの依存パッケージインストール機能を提供します

### 修正
- Linuxビルドが再び利用可能になりました（既知の問題から削除されました）

### 変更
- READMEは明確な初回およびアップデートビルド手順で更新されました

## [0.3.1] - 2026-06-25

### 修正
- GitHub Actionsリリースが正しくEXEファイルを含むようになりました

## [0.3.0] - 2026-06-25

### 追加
- AVIF形式のサポート（静止画とアニメーション）
- アニメーション対応: GIF、AVIF、WebP（APNGはサポートされていません）

### 修正
- スタンドアロンファイルとアーカイブファイル間で同じ名前のキャッシュ衝突を修正
- キャッシュは、ソースファイルがキャッシュされたバージョンより新しい場合に再生成されます

### 既知の問題
- 中央マウスボタンクリックはフルスクリーン切り替えを実行するために軽いマウス移動が必要です
- Linuxビルドは一時的に利用できません
